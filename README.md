## Pinout
<p align="center"><img src="media/pinout.png"><p>

  #### Discharge
  <p>PD4 = U/D</p>
  <p>PD5 = INC</p>
  <p>PD6 = CS</p>
  <p>PD3 = Enable</p>
  
  #### Charge
  <p>PA3 = U/D</p>
  <p>PA2 = INC</p>
  <p>PA1 = CS</p>
  <p>PD1 = Enable</p>
  <p>PB5 = CHRG</p>

  
  #### Buttons
  <p>PC3 = button1</p>
  <p>PB4 = button2</p>
  
  #### ADC 
  <p>PC4 = voltage refernece</p>
  <p>PD2 = battery</p>
  
  #### Display
  <p>PC7 = DIN</p>
  <p>PC6 = CS</p>
  <p>PC5 = CLK</p>
